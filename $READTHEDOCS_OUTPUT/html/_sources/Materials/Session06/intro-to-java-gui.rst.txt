Introduction to Java GUI Programming
====================================

Pemrograman GUI (Graphical User Interface) di Java memungkinkan pengembang untuk membuat aplikasi dengan antarmuka pengguna yang interaktif. 
Java menyediakan beberapa framework untuk membangun GUI, salah satunya yang akan kalian pelajari adalah JavaFX.

JavaFX adalah framework GUI modern yang dirancang untuk menggantikan Swing sebagai toolkit GUI utama untuk Java. JavaFX menyediakan antarmuka pengguna yang lebih kaya dan mendukung berbagai fitur multimedia dan grafik.

- Komponen JavaFX: Button, Label, TextField, TextArea, CheckBox, Combo Box, dan lain-lain.

- Layouts: HBox, VBox, BorderPane, GridPane, StackPane, dan lain-lain.

- Styling: Dukungan untuk CSS untuk styling komponen dan tata letak.

- Animation and Effects: Dukungan untuk animasi dan efek visual.

- Multimedia: Dukungan untuk audio dan video playback.

- FXML: File XML yang digunakan untuk mendeskripsikan antarmuka pengguna, memungkinkan pemisahan logika aplikasi dan tampilan.

Komponen Utama JavaFX

- Stage: Merupakan jendela utama aplikasi JavaFX. Setiap aplikasi JavaFX memiliki setidaknya satu stage.

- Scene: Merupakan wadah untuk semua konten dalam aplikasi. Scene ditempatkan di dalam stage.

- Node: Setiap elemen dalam scene graph adalah node. Node bisa berupa kontrol UI, tata letak, atau bahkan bentuk dasar seperti garis dan lingkaran.

Event Driven Programming II

Setelah mempelajari berbagai cara dalam melakukan event driven programming, kita akan mempelajari apa saja jenis-jenis event yang ada dalam JavaFX. Pada sesi ini, kita akan mempelajari beberapa jenis event yang sering digunakan dalam JavaFX, yaitu:

Mouse Listener

Mouse listener adalah event yang terjadi ketika pengguna melakukan interaksi dengan mouse. Beberapa contoh event yang sering digunakan dalam mouse listener sebagai berikut.

  • Mouse Clicked -> event yang terjadi ketika pengguna melakukan klik pada mouse.

  • Mouse Entered -> event yang terjadi ketika mouse masuk ke dalam area node.

  • Mouse Exited -> event yang terjadi ketika mouse keluar dari area node.

  • Mouse Pressed -> event yang terjadi ketika pengguna menekan tombol mouse.

  • Mouse Released -> event yang terjadi ketika pengguna melepas tombol mouse.

Keyboard Listener

Keyboard listener adalah event yang terjadi ketika pengguna melakukan interaksi dengan keyboard. Beberapa contoh event yang sering digunakan dalam keyboard listener sebagai berikut.

  • Key Pressed -> event yang terjadi ketika pengguna menekan tombol keyboard.

  • Key Released -> event yang terjadi ketika pengguna melepas tombol keyboard.

  • Key Code -> event yang terjadi ketika pengguna menekan tombol keyboard dengan kode tertentu.

  • Key Character -> event yang terjadi ketika pengguna menekan tombol keyboard dengan karakter tertentu.

Item Listener

Item listener adalah event yang terjadi ketika pengguna melakukan interaksi dengan item. Beberapa contoh event yang sering digunakan dalam item listener sebagai berikut.

  • Item State Changed -> event yang terjadi ketika status item berubah.

  • Item Selected -> event yang terjadi ketika item dipilih.

  • Item Deselected -> event yang terjadi ketika item tidak dipilih.

  • Item Clicked -> event yang terjadi ketika item diklik.

  • Item Entered -> event yang terjadi ketika mouse masuk ke dalam area item.

  • Item Exited -> event yang terjadi ketika mouse keluar dari area item.

  • Item Pressed -> event yang terjadi ketika pengguna menekan tombol mouse pada item.

  • Item Released -> event yang terjadi ketika pengguna melepas tombol mouse pada item.

Persiapan Database MySQL
========================

**Langkah 1**. Buka XAMPP Control Panel dan start Apache dan MySQL.

.. image:: /Assets/Session10/Setup/Picture1.png
    :width: 300
    :align: center
.. centered:: Start Apache dan MySQL di XAMPP Control Panel

**Langkah 2**. Buka php my admin di browser dengan klik *Admin* pada MySQL.

.. image:: /Assets/Session10/Setup/Picture2.png
    :width: 300
    :align: center
.. centered:: Akses php my admin

**Langkah 3**. Buat database baru dengan klik *Baru*.

.. image:: /Assets/Session10/Setup/Picture3.png
    :width: 300
    :align: center
.. centered:: Membuat database baru

**Langkah 4**. Beri nama database sesuai keinginan, lalu klik *Buat*.

.. image:: /Assets/Session10/Setup/Picture4.png
    :width: 300
    :align: center
.. centered:: Memberi nama database

**Langkah 5**. Buat dan beri name tabel baru sesuai keinginan, lalu klik *Kirim*.

.. image:: /Assets/Session10/Setup/Picture5.png
    :width: 300
    :align: center
.. centered:: Membuat tabel baru

**Langkah 6**. Buat kolom-kolom pada tabel sesuai keinginan, lalu klik *Simpan*.

.. image:: /Assets/Session10/Setup/Picture6.png
    :width: 300
    :align: center
.. centered:: Membuat kolom pada tabel

.. image:: /Assets/Session10/Setup/Picture7.png
    :width: 300
    :align: center
.. centered:: Hasil kolom pada tabel

Pada tahap ini, database MySQL sudah siap digunakan.

.. note:: 

    Pastikan database yang dibuat sudah sesuai dengan kebutuhan aplikasi yang akan dibuat. Pastikan juga project anda sudah mengimport library mysql-connector-java.

    .. image:: /Assets/Session10/Setup/Picture8.png
        :width: 300
        :align: center
